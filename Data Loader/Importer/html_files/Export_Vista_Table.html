'Export VistA Tables' Menu Options:

Description:

Gender

Fetches, saves, and displays (via notepad) allowable gender/sex selection values from the VistA server. This list is used to constrain input entries into the VistA system. In the Data_Entry_Template.xls data input template, the list should match the values in the Tables worksheet; Gender column. This list is saved in [APPDATA]\Importer\Gender.txt.
