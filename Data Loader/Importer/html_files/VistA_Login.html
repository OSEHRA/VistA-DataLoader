'VistA Login' Menu Options:

Description:

Connect

Opens the Importer client's broker connection with the VistA server. Derives connection information from values applied in the Settings menu option (below).

Disconnect

Closes the Importer client's broker connection with the VistA server.

Settings

Launches Connection Settings window. The Connection Settings window allows the user to edit the server (hostname/IP) and port information for the target VistA system.
